# Ekonomi Laboratuvarı (Prototype)

Gençlerin finansal okuryazarlığını desteklemek için **oyunlaştırılmış interaktif ders prototipi**.

## Modüller (MVP)
- Enflasyon Laboratuvarı (alım gücü / sepet)
- Faiz & Kredi Laboratuvarı (taksit / toplam ödeme / bileşik)
- Merkez Bankası Karar Odası (basitleştirilmiş trade-off sim)

> Not: Modeller eğitim amaçlı basitleştirilmiştir.

## Çalıştırma
```bash
npm install
npm run dev
```

## Tech
- Vite + React + TypeScript
- MUI
- Recharts
